API
(1)
登陆：
URL：
method：POST
参数：
{ 
	code：string                      微信wx.login()获取的code

	encryptedData: string       用户敏感信息加密数据

	iv: string                             加密向量
}
response：

{
	 code：string                         响应码
	 msg:    string                         响应信息
	uid：
	uNickName:		用户昵称
	uavaUrl:[			用户头像
	               		]
}

	identity:
